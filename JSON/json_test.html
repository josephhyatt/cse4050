What is JSON? (check out this video overview)

JSON stands for Java Script Object Notation.
JSON is a syntax for storing and exchanging data. JSON is text, written with JavaScript object notation.

It is a language independent text format that holds the following data types:

String
Number
Boolean Function
Array (ordered list of differing values) and
Object (unordered list of various values i.e. name types).

Essentially JSON is a wire protocol that helps serialize and de-serialize data.
In layman terms, you can think of JSON as an language interpreter between two people who don't speak the same language.
